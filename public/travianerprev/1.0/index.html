Travianer - Version 1.0

Published on 2024-07-15

This is a Unity game project made with Unity 2022.3.13f1. This project was done with the collaboration of Viktória Horváth, who helped in many graphical, and coding assets of the project. The project is based on the web browser, war strategy game, Travian by Travian Games GmbH. Assets are reused from the original Travian project, and we do not own any of them. They are the intellectual property of Travian Games GmbH. The game was originally made for PC, and this WebGL version below is not optimized, and some assets might have different scaling because of it. I recommend playing it in 1920x1080 fullscreen.

Note: all other players in the game are controlled by bots.

Note: If you choose full screen, you might have to click again in the game screen for it to take effect.

This content does not resize on smaller browser widths. Try using the fullscreen button below (after loading the game).
Unity WebGL builds are not supported on mobile devices.
