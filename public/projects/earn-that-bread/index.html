Earn That Bread

Published on 2022-08-10

This game is an entry for the Mini Jam 94: Baking game jam. The game jam came with a limitation: You can only use 2 buttons.

The GitHub repository of the game can be found here:

  • GitHub repository

The goal of this game is to bake an amount of bread that was ordered by the king. You need to get ingredients to bake bread, however on the way to collect them, you might run into some trouble. The two buttons you can use are the two mouse buttons.

Unity Engine 2020.3.19f1 was used. Credits for Bensound.com for the music.
